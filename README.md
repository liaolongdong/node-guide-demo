## node具体用法
- fs模块（文件的读写操作）
- path模块（路径各种方法）
- process对象（全局对象）
- Buffer对象（二进制）
- events模块（事件）
- stream接口（数据流）
- child_process模块（用于新建子进程）
- http模块（主要用于搭建HTTP服务）
- assert模块（主要用于断言）
- cluster模块（允许设立一个主进程和若干个worker进程）
- os模块（提供与操作系统相关的方法）
